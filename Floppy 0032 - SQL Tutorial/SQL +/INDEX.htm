Subject

This is the home page for the maintainer of the Oracle FAQ Web Pages and contains jump points to the FAQ and the related FTP archive area. Parts of this FAQ are currently under major revision and will appear RSN at a new address to be announced. The revisions are not currently visible from this site. If anyone has comments email me here. BTW - My home address (physical) will be changing to. Busy Busy

Intended Audience

Anyone using, or thinking about using, the Oracle Relational Database Management System or related tools.

Major Jump Points

  • Contents page of the Oracle FAQ
    This is where you probably want to go if you have got this far, have a reasonable connection, and want to be able to follow links to other machines or the archive site.
  • Gets all the pages in one hit
    All the local pages (which use relative links to each other rather than absolute ones, so they should work on your local machine except for links to other sites or ftp archive area) are zipped up into one file. Good if you have limited access to the Net or if you find the link here to Oz is slow.
  • Mirrors of these Pages
    You might find it easier to reference a local mirror of these pages. Some will have just the pages, some will have some of the FTP archive. Here they are:
    • Unknown as yet
  • FTP archive area
    The FTP archives associated with these pages are at ftp://ftp.bf.rmit.edu.au/pub/Oracle. This is useful to browse. It contains utilities, demons and a couple of things from www.oracle.com.

    The tools directly from Oracle under pub/Oracle/Sites/Oracle should NOT be accessed if you are restricted from doing so under U.S. laws, or if you want to be guaranteed the latest versions. They are put here mainly for Australians to avoid overloading the links to the US.

    A (hopefully recent) list of the contents of the archive (without descriptions) that contains hot spots linking to the various files is in FTP archive list page.
  • Oracle Corporation Links
    These are not authorized by the author of the Oracle FAQ pages ;-).
    • Web Pages
      A guided tour through Oracle's products, with news, FAQ, etc.
    • FTP Site
      If you do not like constantly renegotiating a connection, use FTP and go to ftp://ftp.oracle.com
  • Author's Home Page
    If you happen to be interested in the personal details of David T. Bath, the author of these FAQ pages, go to http://www.bf.rmit.edu.au/~dtb
